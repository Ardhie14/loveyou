### 🌷 Demo
👉 [View on GitHub Pages](https://ardhie14.github.io/loveyou/)

### 💞 How to Use
1. Upload files to your GitHub repository named `loveyou`
2. Go to <b>Settings → Pages</b>
3. Enable GitHub Pages for the `main` branch
4. Send your special link 💌
